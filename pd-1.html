AI-Powered E-commerce Platform

Python(Django) + LLM-powered + Vanilla JS

Project Overview

This AI-powered e-commerce solution enhances product discovery using natural language processing. Built with Django and Google's Gemini AI, the backend interprets user-friendly search queries and returns precise hoodie recommendations via JSON. A custom filtering engine matches products based on dynamic attributes like price, color, size, and material.

The frontend, developed using plain HTML, CSS, and JavaScript, sends AJAX-based POST requests for seamless user experience without page reloads. By avoiding heavy frameworks, the platform stays lightweight while still delivering powerful AI functionality. The integration of conversational AI reduced search friction and improved user engagement and conversion performance.
